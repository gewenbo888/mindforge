# MindForge · 心智锻造

A structured system for cognitive growth, personality development, and lifelong self-transformation. Bilingual (EN / 中文), dark/light themes, with nine modules: Cognitive Architecture Atlas, Personality Engine, Emotion Wheel + regulation scripts, Decision Lab with bias catalog, Habit & Dopamine system, Metacognition pyramid, Attachment patterns, AI Cognitive Coach, and a multi-year Growth Timeline.

## Links

- **Live:** [mindforge.psyverse.fun](https://mindforge.psyverse.fun)
- **Vercel:** https://mindforge.vercel.app
- **GitHub:** https://github.com/gewenbo888/mindforge

## Stack

Plain HTML, CSS, vanilla JavaScript. No build step. Bilingual via `[data-lang]` + CSS attribute selectors. Dark/light via `[data-theme]`. Custom SVG diagrams for the cognitive atlas, emotion wheel, habit loop, learning pyramid, and growth timeline. Fonts: Instrument Serif (display) + Geist + Geist Mono (Google Fonts).
